# Common scenarios

Each page below is a recipe for a recurring situation that does not
fit cleanly into a single lifecycle step. Pick the page that matches
your situation.

- [I just got assigned a new bug](./new-bug.md)
- [I have several unrelated bugs to fix](./parallel-bugs.md)
- [I want to do exploratory work](./exploratory-work.md)
- [I think I have seen this bug before](./find-prior-art.md)
- [Something about the workflow feels broken](./debug-workflow.md)
- [I need to tweak a hook](./tweak-a-hook.md)
- [I need to add a path-scoped rule](./add-path-rule.md)

If your situation does not match any page, the lifecycle pages
([Open a session](../open-a-session.md),
[Claim a bead](../claim-a-bead.md),
[Finish a bead](../finish-a-bead.md),
[Stop a session](../stop-a-session.md)) cover the linear flow.
