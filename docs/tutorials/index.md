# Tutorials

Tutorials are the *acquisition + action* quadrant: guided narratives
that teach you loom by doing real work in a controlled, reproducible
way. You won't pick the path; the tutorial will. You'll learn by
following along, watching the primitives fire, and seeing each
hand-off between beads, MemPalace, and the skills.

> **What tutorials are not.** They are not how-to guides (those
> assume you already know what you want to do) and not reference
> material (those describe the surface, not the journey).
